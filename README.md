Spotify Music Insights Dashboard

An interactive Power BI dashboard analyzing Spotify music data to explore trends in artists, genres, popularity, and release years.
The dashboard provides insights into music listening patterns and highlights the characteristics of popular tracks on Spotify.

Dashboard Highlights

Top artists and most popular tracks analysis

Genre distribution across songs

Popularity score analysis

Release year trends of songs

Interactive filters for artist, genre, and year

Insights

Some artists consistently appear among the most popular tracks.

Certain genres dominate the Spotify music library.

Modern songs generally show higher popularity scores.

Music releases have increased significantly in recent years.

Tools Used

Power BI – Dashboard design and visualization

Excel / CSV – Data cleaning and preprocessing

Git – Version control

GitHub – Project hosting using GitHub

How to Use

Download the .pbix file from this repository.

Open it using Power BI Desktop.

Explore the dashboard using interactive visuals and filters.

Dataset Source

Spotify dataset used for educational data analysis.
